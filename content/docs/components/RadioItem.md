# Radio Item

`RadioItem` is a visual only element used it [RadioGroup](/docs/components/radiogroup), [Menu](/docs/components/menu) and [DropdownMenu](/docs/components/dropdownmenu)

___

## Anatomy

```vue
<template>
  <RadioItem.Root>
    <RadioItem.Indicator />
  </RadioItem.Root>
</template>
```
