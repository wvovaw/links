# Dropdown Menu

`DropdownMenu` is a set of actions of functions displayed to users after they trigger a button.

___

## Anatomy

```vue
<template>
  <DropdownMenu.Root>
    <DropdownMenu.Trigger />
    <DropdownMenu.Content>
      <DropdownMenuLabel />
      <DropdownMenuSeparator />
      <DropdownMenu.Item />
    </DropdownMenu.Content>
  </DropdownMenu.Root>
</template>
```
